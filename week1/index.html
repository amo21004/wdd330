Notes pertaining to Mobile UX

  • Use a Mobile First approach when creating the UX
  • All designs need to be cross-device compatible
  • Avoid situations that require "Responsive Retrofitting"
  • Reduce steps as much as possible that are required to complete a form
  • Focus on loading speed as many users use mobile data while surfing using a mobile device
  • Optimize images to use modern formats such as WEBP and minify the HTML, CSS, and JS
  • Considering the importance of the primary navigation menu, make sure it is highly accessible on mobile devices
  • Use correct field types on forms. For example, if a certain field requires a number, use the type "number" instead of "text"
  • Whenever using placeholders as labels, make sure to implement floating labels
  • Use ads sparingly and place them strategically

Code to fetch data from JSON file and populate the ordered list


      
document.addEventListener("DOMContentLoaded", function () {
  const toc_url = "json/toc.json";

  fetch(toc_url)
    .then((response) => response.json())
    .then((json_object) => {
      const toc_list = document.querySelector("#toc");

      for (item of json_object.links) {
        const li_element = document.createElement("li");

        const anchor_element = document.createElement("a");

        anchor_element.href = item.url;

        anchor_element.innerText = item.label;

        li_element.appendChild(anchor_element);

        toc_list.appendChild(li_element);
      }
    });
});
