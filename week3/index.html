Object Methods: this

  • A special keyword named "this" is used to access the internal properties and methods of an object

Ch5: Objects

  • If it's not one of the six primitive data types then it must be an object. This includes objects themselves as well as arrays and functions.
  • Objects are copied by reference. So, if variable X holds an object and variable X is assigned to a variable named Y, then both the variables X and Y point to the exact same object. Y does not hold a copy of the object that was heled by X. It's just a reference to the original object.

Ch6: Document Object Model

  • The DOM is a representation of all the connected nodes using a tree like structure
  • The DOM can be accessed using a built in object called "document"

Ch7: Events

  • Events are a part of the DOM and they are responsible for handling the user's interaction with the page
  • Events make JavaScript a non-blocking language via the use of event listeners
