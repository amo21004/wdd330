Ch8: Forms

  • Form fields have events unique only to them
  • There are various types of fields to choose from. From generic text, to email addresses, phone numbers, dates, and more.

Ch12: Object Oriented Programming

  • Objects in JavaScipt can be created using "Constructor Functions" or using "ES6 Class Declarations"
  • The way objects and classes are dealt with in JavaScript is significantly different from other languages. This is why it's referred to as a prototype-based language.

Ch15: Modern Javascript

  • Modules help create reusable code.
  • Modules can be used to write code that's loosely coupled.
  • A module should ideally be responsible for doing one thing only.
