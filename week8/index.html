Ch8: Transforms and Transitions

  • The transform property is used to translate, rotate, scale, and skew elements.
  • The order of transforms matters.
  • The transition property can be used to create delayed effects.
  • Animations in CSS allow creation of steps using keyframes.
  • Various properties including transform is used for creating animations

Ch12: Canvas, SVG, and Drag and Drop

  • When creating a canvas element, the width and height need to be provided explicitly using the HTML attributes. Assigning a width and height using CSS to a canvas element does something very different.
  • After creating a canvas element, we need to create a context for it using JavaScript. The context is used for actual drawing.
  • The coordinate system used by canvas is different from the Cartesian coordinate system.
  • SVG elements that have JavaScript embedded won't be executed if the SVG is loaded using the img tag. To overcome this, either the SVG must be embedded directly into the HTML or the object element should be used.
