Ch9: The Window Object

  • Inside a browser, the JavaScript global object is the "window" object.
  • Global variables and functions are stored as property of the window object.
  • The Browser Object Model stores properties and methods that contain information about the browser, the window (for example, the URL of the current page, etc.) and the screen.
  • The window object is different from the document object. The document object is stored as a property of the window object. However, there is no need to refer to it as window.document. Just document will work fine.
  • The document object represents the page that is loaded inside a window.
  • Cookies are useful for preserving the state considering that the HTTP is stateless.
  • setTimeout and setInterval are examples of two timing functions.

Ch14: HTML5 APIs

  • The HTML5 API includes the Web Storage API, Geolocation API, Notification API, among others.
  • The Web Storage API can be used like Cookies. In comparison to Cookies, the Web Storage API provides more space.
